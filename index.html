Help & FAQ

How does it work?

VoxelHost creates isolated "rooms" on the VoxelHost game server. Each room is a separate server — players in different rooms can't see each other. When you create a server, you get a unique wss:// link that players use to join.

How to join a server

  1. Open VoxelCraft
  2. Click Multiplayer
  3. Click "+ Add Server"
  4. Paste the wss:// link from VoxelHost
  5. Click Join!

Console Commands

/list — list players · /kick <name> — kick · /say <msg> — broadcast · /info — server info · /help — all commands
